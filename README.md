# 3DShop — статический каталог 3D-печатных моделей

Миниатюрный статический сайт без backend для демонстрации и заказа 3D-печатных моделей. Данные каталога подгружаются из `data/products.json`, связь и оплата проходят в Telegram.

## Страницы
- **index.html** — краткое описание студии, CTA в каталог и для кастомного заказа.
- **catalog.html** — список товаров с поиском, фильтрами (категория, цена, материал) и сортировками.
- **product.html** — карточка товара с галереей, характеристиками и кнопками заказа.
- **contact.html** — условия работы, оплата, доставка и ссылки на мессенджеры.

## Как открыть локально
1. Запустите любой статический сервер из корня проекта (пример для Python):
   ```bash
   python -m http.server 8000
   ```
2. Перейдите в браузере на `http://localhost:8000`.

> Прямая загрузка файлов с диска (`file://`) может блокировать `fetch` для `data/products.json`, поэтому нужен простой сервер.

## Структура
- `data/products.json` — список товаров.
- `assets/css/style.css` — единый минималистичный стиль и адаптив.
- `assets/js/common.js` — общий код (инициализация модального окна, загрузка каталога).
- `assets/js/catalog.js` — фильтрация, сортировка и отрисовка карточек каталога.
- `assets/js/product.js` — страница товара, галерея и вывод характеристик.
- `assets/img/products/*` — заглушки изображений товаров.
